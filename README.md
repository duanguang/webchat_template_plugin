## Getting started

```bash
# node -v 18.12.1

# install @tarojs/cli
```bash
npm install -g @tarojs/cli@3.6.32
```

# install dependency
yarn install

# develop
yarn dev:weapp
```


## Build
```bash
# build for test environment
npm run build:weapp

# build for production environment
npm run build:weapp
```
